String Type

The string type in Sour Lang represents a sequence of Unicode characters. String literals are enclosed in double quotes.

Note: Sour Lang currently supports strings with a maximum length of 255 characters.

Properties

string.len

Returns the length of the string as a byte type.

var my_string = "Hello"
var length = my_string.len // length will be 5 (byte type)

Character Access

Individual characters within a string can be accessed using square bracket notation (indexing). The index must be a byte value. This operation returns a char type.

var my_string = "Sour"
var first_char = my_string[0] // first_char will be 'S' (char type)
var second_char = my_string[1] // second_char will be 'o' (char type)

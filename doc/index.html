Sour Editor

A lightweight, mobile-first JavaScript IDE that runs directly in your browser. Designed for simplicity and efficiency, Sour Editor provides a seamless coding experience on the go.

Features:

  • File Management: Create, save, load, and delete files using local storage. Manage multiple open files with intuitive tabs.
  • Run Code: Execute your Sour Language code directly within the editor and view real-time output and errors in a dedicated console.
  • Enhanced Syntax Highlighting: Enjoy advanced syntax highlighting for Sour Language, including smart bracket matching with depth-based coloring and current line highlighting.
  • Intelligent Autocompletion: Get smart code completion suggestions for keywords and functions, complete with function signatures to speed up your coding.
  • Interactive Error Tooltips: Debug with ease thanks to detailed error messages displayed directly in the editor.
  • Contextual Documentation: Access documentation for autocompletion suggestions, including function parameters and return types, right when you need it.

Built-in Data Types

Sour Lang provides a set of built-in data types. Here are the fundamental types currently available:

char

The char type represents a single Unicode character. Character literals are specified with a single quote.

var my_char = 'a'

byte

The byte type represents an 8-bit unsigned integer (0 to 255). Byte literals are specified with a 'b' prefix.

var my_byte = 255

int

The int type represents a signed integer. This type is not yet implemented.

string

The string type represents a sequence of characters. String literals can be created with double quotes.

var my_string = "Hello, World!"

bool

The bool type represents a boolean value, which can be either true or false.

var my_bool = true

Future Data Types

The following data types are planned for future releases:

  • array: A collection of elements of the same type.
